01 · 서비스 개요

어떤 서비스인가요?

보팅 라이프 플랫폼은 보트 · 페리 · 요트 등 해양 레저를 즐기는 사람들을 위한 모바일 앱입니다. 서버는 앱이 정상적으로 작동하도록 데이터를 저장하고, 사용자를 인증하고, 결제를 처리하는 역할을 합니다.

📱
모바일 앱 (프론트)
사용자가 직접 보는 화면. Flutter로 제작하며 iOS · Android 모두 지원합니다. 다른 팀원 담당.
🖥️
서버 (백엔드)
앱의 요청을 받아 데이터를 처리하고 저장합니다. 본 기획안의 주요 대상입니다.
🗄️
데이터베이스
회원 정보, 게시글, 예약 · 결제 내역 등 모든 데이터를 안전하게 보관합니다.

💡
역할 분리 원칙
앱(프론트)은 사용자 눈에 보이는 것, 서버(백엔드)는 눈에 보이지 않는 데이터 처리를 담당합니다. 예를 들어 사진 업로드 시 — 앱이 사진을 클라우드에 올리고, 서버는 그 사진 주소를 받아 데이터베이스에 기록하는 방식으로 역할을 나눕니다.

03 · 로그인 · 회원가입

어떻게 로그인하나요?

별도의 아이디 · 비밀번호 없이 구글 또는 카카오 계정으로 간편하게 가입하고 로그인할 수 있습니다.

로그인 흐름 소셜 로그인
①
로그인 버튼 클릭
구글 또는 카카오 선택
②
소셜 계정 인증
해당 앱에서 처리
③
인증 토큰 발급
신분증 역할
④
서버 검증
토큰이 진짜인지 확인
⑤
회원 확인
신규면 자동 가입
⑥
로그인 완료
앱 진입
🔑
토큰(Token)이란?
로그인 후 서버가 발급하는 디지털 신분증입니다. 앱은 이후 모든 요청에 이 토큰을 함께 보내고, 서버는 토큰을 확인해 "진짜 우리 회원인지" 검증합니다. 토큰이 없거나 유효하지 않으면 서버가 요청을 거부합니다.
04 · 주요 기능

서버가 제공하는 기능

서버는 앱의 요청을 받아 아래 기능들을 처리합니다. 기능별로 어떤 동작을 지원하는지 정리했습니다.

👤
회원 관리
프로필 조회 및 수정, 닉네임 · 소개 · 프로필 사진 변경을 지원합니다. 다른 사용자의 프로필도 조회할 수 있습니다.
📸
SNS 피드 · 게시글
해양 레저 경험을 사진 · 글로 공유합니다. 게시글 작성 · 수정 · 삭제, 피드 조회(최신순), 특정 스팟 게시글 모아보기를 지원합니다.
💬
댓글 · 좋아요
게시글에 댓글을 달고 좋아요를 누를 수 있습니다. 본인 댓글만 수정 · 삭제 가능하며, 좋아요는 한 번 더 누르면 취소됩니다.
🗺️
해양 레저 스팟
보트 · 페리 · 요트 등 스팟 정보를 제공합니다. 지도 기반으로 현재 위치 주변 스팟을 반경 검색하고, 상세 정보를 확인할 수 있습니다.
📅
예약
원하는 스팟을 날짜 · 시간에 맞춰 예약합니다. 예약 내역 조회 및 취소가 가능합니다.
💳
결제 (토스페이먼츠)
예약 후 토스페이먼츠로 안전하게 결제합니다. 서버에서 결제 금액을 이중 검증하여 위변조를 방지하고, 결제 취소도 지원합니다.
📍
위치 · 트래킹
사용자의 현재 위치를 기록하고 이동 이력을 저장합니다. 항해 경로 기록 등 해양 레저 활동에 특화된 기능을 지원합니다.
🖼️
이미지 업로드
사진은 앱에서 클라우드(Firebase)에 직접 업로드되고, 서버는 해당 사진 주소만 받아 저장합니다. 서버 부하를 줄이는 효율적인 방식입니다.

06 · 보안

어떻게 안전하게 보호하나요?

사용자 데이터와 결제 정보를 보호하기 위해 여러 겹의 보안 장치를 적용합니다.

1
모든 통신 암호화 (HTTPS)
앱과 서버 간 모든 데이터를 암호화해 전송합니다. 중간에 누군가 가로채도 내용을 볼 수 없습니다.
2
로그인 토큰 검증
모든 API 요청에 로그인 토큰이 필요합니다. 토큰이 없거나 유효하지 않으면 요청이 자동으로 거부됩니다.
3
본인 인증 (권한 검증)
게시글·댓글 수정 및 삭제는 작성자 본인만 가능합니다. 다른 사람이 타인의 글을 수정할 수 없습니다.
4
결제 금액 이중 검증
결제 완료 직전, 서버가 토스페이먼츠에 직접 금액을 확인합니다. 앱에서 금액을 임의 변조해도 서버에서 차단됩니다.
5
민감 정보 별도 관리
각종 API 비밀키는 코드에 포함시키지 않고 별도 보안 저장소(AWS Secrets Manager)에 보관합니다. 소스코드가 외부에 노출되어도 키는 유출되지 않습니다.
6
앱 전용 접근 제한 (CORS)
우리 앱에서 보낸 요청만 수락합니다. 외부 웹사이트나 다른 앱에서 무단으로 API를 호출하는 것을 방지합니다.
07 · 인프라 · 배포

서버는 어디서 운영되나요?

Amazon Web Services(AWS) 클라우드 위에서 서버와 데이터베이스를 운영합니다.

☁️
서버 컴퓨터 (EC2)
실제 서버 프로그램이 실행되는 클라우드 컴퓨터입니다. 24시간 365일 항상 켜져 있어 언제든 요청에 응답합니다.
🗄️
데이터베이스 서버 (RDS)
회원 · 게시글 · 예약 · 결제 데이터를 저장하는 전용 데이터베이스 서버입니다. 자동 백업 기능을 제공합니다.
🌐
도메인 · 트래픽 분산
사용자 요청을 서버로 안전하게 연결하고, 사용자가 많아질 경우 자동으로 부하를 분산합니다.
🔒
보안 키 보관소
결제 API 키, 데이터베이스 비밀번호 등 민감한 정보를 별도 보안 금고에 보관합니다.

📦
연동 외부 서비스 요약
소셜 로그인(구글·카카오), 이미지 저장소(Firebase), 결제(토스페이먼츠), 지도(Google Maps) 등 각 분야 전문 서비스를 연동하여 안정성을 높이고 개발 기간을 단축합니다.
08 · 데이터 관리

어떤 데이터를 저장하나요?

서비스 운영에 필요한 데이터를 항목별로 분류해 저장합니다. ERD(상세 구조도)는 별도 문서로 정리 예정입니다.

ERD 상세 문서 — 프론트엔드 설계 확정 후 업데이트 예정
저장 항목 포함 정보 비고
👤 회원 정보 닉네임, 이메일, 프로필 사진 주소, 로그인 방식(구글/카카오) 비밀번호 미저장 (소셜 로그인)
📸 게시글 글 내용, 연결된 스팟, 작성 위치 좌표, 작성일 이미지는 URL만 저장
🖼️ 게시글 이미지 이미지 클라우드 주소(URL), 순서 실제 이미지는 Firebase에 저장
💬 댓글 댓글 내용, 작성자, 게시글 연결 정보
❤️ 좋아요 누른 사용자, 게시글 연결 정보 중복 방지
🏖️ 레저 스팟 스팟 이름, 종류(보트/페리/요트), 위치 좌표, 설명 관리자만 등록 가능
📅 예약 내역 예약자, 스팟, 날짜·시간, 인원, 상태(예약중/취소)
💳 결제 내역 예약 연결 정보, 결제 금액, 상태(완료/취소) 카드번호 등 민감정보 미저장
📍 위치 이력 사용자 위치 좌표, 기록 시각 항해 경로 추적용

🔐
개인정보 보호 원칙
비밀번호는 저장하지 않습니다 (소셜 로그인 사용). 결제 카드번호 등 금융 민감 정보는 토스페이먼츠에서만 관리하며, 우리 서버에는 저장되지 않습니다.

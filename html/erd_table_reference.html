USER
서비스에 가입한 회원 정보를 저장하는 핵심 테이블. 인증·프로필·누적 통계를 포함합니다.
컬럼명타입키설명
iduuidPK회원 고유 식별자. 내부 참조에 사용됩니다.
emailstring로그인에 사용되는 이메일 주소. UNIQUE 제약이 필요합니다.
password_hashstring비밀번호를 bcrypt 등으로 해시한 값. 평문 저장 절대 금지.
member_codestring사람이 읽기 쉬운 회원 코드 (예: USR-00123). 고객센터 식별용.
birth_datedate생년월일. 연령 인증이나 마케팅 세그먼트에 활용합니다.
phonestring전화번호. SMS 인증·알림 발송에 사용됩니다.
joined_attimestamp회원 가입 일시. 자동 설정 (DEFAULT NOW()).
profile_imagestring프로필 이미지 URL 또는 파일 경로.
languagestring선호 언어 코드 (kor / eng). UI 언어 전환에 사용.
accumulated_sailing_hoursfloat누적 항해 시간. VESSEL_LOG에서 집계하거나 트리거로 갱신.
accumulated_distancefloat누적 항해 거리(km). VESSEL_LOG에서 집계하거나 트리거로 갱신.
NOTIFICATION_SETTING
회원별 알림 수신 설정. USER와 1:1 관계이며, 채널별 on/off를 관리합니다.
컬럼명타입키설명
iduuidPK설정 고유 식별자.
user_iduuidFKUSER.id 참조. 어떤 회원의 설정인지 나타냅니다.
push_enabledbool앱 푸시 알림 수신 여부.
email_enabledbool이메일 알림 수신 여부.
sms_enabledboolSMS 알림 수신 여부.
LANGUAGE_SETTING
언어 설정을 별도 테이블로 분리해 다국어 확장을 유연하게 지원합니다.
컬럼명타입키설명
iduuidPK설정 고유 식별자.
user_iduuidFKUSER.id 참조.
localestring언어 로케일 코드 (예: ko-KR, en-US).
VESSEL_LOG
회원의 항해 기록을 저장합니다. 누적 시간·거리 집계의 원천 데이터입니다.
컬럼명타입키설명
iduuidPK항해 기록 고유 식별자.
user_iduuidFKUSER.id 참조. 누구의 항해인지 나타냅니다.
started_attimestamp항해 시작 일시.
ended_attimestamp항해 종료 일시.
distance_kmfloat이번 항해에서 이동한 거리(km).
duration_hoursfloat이번 항해 소요 시간(시간 단위).
route_descriptionstring항해 경로나 메모를 자유 텍스트로 기록합니다.
PRODUCT
예약 가능한 항해 상품 정보입니다. 태그는 PRODUCT_TAG에 분리 저장됩니다.
컬럼명타입키설명
iduuidPK상품 고유 식별자.
namestring상품명.
descriptionstring상품 상세 설명.
duration_minutesint상품 이용 소요 시간(분).
addressstring출발/집결 장소 주소.
base_pricedecimal기본 이용 요금. 인원 수에 따라 결제 시 계산됩니다.
provided_itemsstring제공 물품 목록 (예: 구명조끼, 낚싯대 등).
avg_ratingfloat리뷰 별점 평균. REVIEW 집계값으로 주기적으로 갱신합니다.
PRODUCT_TAG
상품에 붙은 해시태그를 1행 1태그로 저장합니다. 검색·필터링에 활용됩니다.
컬럼명타입키설명
iduuidPK태그 행 고유 식별자.
product_iduuidFKPRODUCT.id 참조.
tagstring태그 문자열 (예: #낚시, #석양크루즈).
RESERVATION
회원이 상품을 예약한 내역입니다. 결제와 1:1로 연결되며, 리뷰의 기준 단위가 됩니다.
컬럼명타입키설명
iduuidPK예약 고유 식별자.
user_iduuidFKUSER.id 참조. 예약한 회원.
product_iduuidFKPRODUCT.id 참조. 어떤 상품을 예약했는지.
payment_iduuidFKPAYMENT.id 참조. 이 예약에 연결된 결제.
reserved_attimestamp예약 일시 (실제 이용 날짜+시간).
seat_numberstring배정된 좌석 번호.
headcountint예약 인원 수.
statusstring예약 상태 (pending / confirmed / cancelled / completed).
PAYMENT
결제 내역을 기록합니다. 예약과 1:1로 연결되며, 환불·취소 이력도 status로 관리합니다.
컬럼명타입키설명
iduuidPK결제 고유 식별자.
user_iduuidFKUSER.id 참조. 결제한 회원.
product_iduuidFKPRODUCT.id 참조. 결제 대상 상품.
amountdecimal실 결제 금액.
methodstring결제 수단 (카드 / 계좌이체 / 간편결제 등).
paid_attimestamp결제 완료 일시.
statusstring결제 상태 (paid / refunded / failed).
REVIEW
실제 예약을 완료한 회원만 리뷰를 작성할 수 있도록 reservation_id로 검증합니다.
컬럼명타입키설명
iduuidPK리뷰 고유 식별자.
user_iduuidFKUSER.id 참조. 리뷰 작성자.
product_iduuidFKPRODUCT.id 참조. 어떤 상품에 대한 리뷰인지.
reservation_iduuidFKRESERVATION.id 참조. 허위 리뷰 방지를 위한 예약 검증.
ratingint별점 (1~5). PRODUCT.avg_rating 집계에 사용됩니다.
contentstring리뷰 본문 텍스트.
created_attimestamp리뷰 작성 일시.
POST
회원이 작성하는 게시글입니다. 해시태그는 POST_HASHTAG, 댓글은 COMMENT에 분리 저장됩니다.
컬럼명타입키설명
iduuidPK게시글 고유 식별자.
user_iduuidFKUSER.id 참조. 게시글 작성자.
titlestring게시글 제목.
bodystring게시글 본문.
posted_attimestamp게시글 등록 일시.
like_countint관심(좋아요) 수. 별도 LIKE 테이블로 분리하면 중복 방지 가능.
POST_HASHTAG
게시글에 붙은 해시태그를 행 단위로 저장합니다.
컬럼명타입키설명
iduuidPK태그 행 고유 식별자.
post_iduuidFKPOST.id 참조.
tagstring태그 문자열 (예: #항해일지).
COMMENT
게시글에 달리는 댓글입니다.
컬럼명타입키설명
iduuidPK댓글 고유 식별자.
post_iduuidFKPOST.id 참조.
user_iduuidFKUSER.id 참조. 댓글 작성자.
contentstring댓글 내용.
created_attimestamp댓글 작성 일시.
REPORT
게시글 신고 내역입니다. 운영자가 처리 여부를 is_resolved로 관리합니다.
컬럼명타입키설명
iduuidPK신고 고유 식별자.
post_iduuidFKPOST.id 참조. 신고된 게시글.
reporter_iduuidFKUSER.id 참조. 신고한 회원.
reasonstring신고 사유 텍스트.
reported_attimestamp신고 접수 일시.
is_resolvedbool운영자 처리 완료 여부 (false = 미처리, true = 처리됨).
INQUIRY
고객센터 문의 내역입니다. 운영자 답변과 처리 여부를 함께 관리합니다.
컬럼명타입키설명
iduuidPK문의 고유 식별자.
user_iduuidFKUSER.id 참조. 문의한 회원.
contentstring문의 내용.
created_attimestamp문의 등록 일시.
answerstring운영자 답변 내용. 미답변 시 NULL.
is_answeredbool답변 완료 여부 (false = 대기중, true = 답변완료).
